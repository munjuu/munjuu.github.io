# munjuu.github.io

My Personal Page: https://munjuu.github.io/

## Requirements
- Install [NodeJS](https://nodejs.org/)
```bash
brew install npm
```
- Install [gulp](https://github.com/gulpjs/gulp/blob/master/docs/getting-started.md)
```bash
npm install --global gulp
```

## Running the blog in local
- Run on local
```bash
gulp
```

- Check the url
http://localhost:3000/
